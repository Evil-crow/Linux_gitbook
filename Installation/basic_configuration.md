**设置语言 -> 选择安装源 -> 安装套件 -> 用户名/密码**

**其中要十分小心,RedHat/CentOS默认安装是没有桌面环境的最小安装版,需要进行调整**

**3. 选择时区**

![选择时区](http://www.qiniu.evilcrow.site/Linux_virtual_box_time_desc.png)

![键盘布局](http://www.qiniu.evilcrow.site/Linux_virtual_box_keybord.png)
